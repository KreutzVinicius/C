					Agendamento de Tarefas
A aplicação possui um menu como o abaixo:
|-----------------------------------------|
|            CADASTRO DE TAREFA           |
|_________________________________________|
|             MENU DE OPÇÕES              |
|-----------------------------------------|
| 1 -INCLUIR NOVA TAREFA                  |
| 2 -VISUALIZAR TAREFAS CADASTRADAS       |
| 3 -EXCLUIR TAREFA                       |
| 4 -EDITAR TAREFA                        |
| 5 -CARREGAR ARQUIVO DE TAREFAS          |
| 6 -GERAR ARQUIVO DE TAREFAS             |
| 7 -COMPUTAR AGENDA DO DIA               |
| 8 -SALVARA AGENDA DO DIA                |
| 0 –SAIR                                 |
|-----------------------------------------|
Para utiliza-lo basta digitar em seu teclado o numero da opçao que deseja escolher,sendo suas funções:
1- adiciona uma nova tarefa na lista de tarefas;
2- imprime todas as tarefas diarias ou todas suas tarefas;
3- imprime todas tarefas cadastradas e pede para voce escolher o id de uma para exclui-la;
4- imprime todas tarefas cadastradas e pede para voce escolher o id de uma para edita suas informações;
5- carrega o arquivo "arq.txt" e cadastra todas tarefas na sua lista de tarefas;
6- apaga a informação existente no arquivo "arq.txt" e escreve a sua lista de tarefas atual;
7- calcula quais tarefas estão com a deadline mais proxima e quantas tarefas voce pode fazer em um periodo de 8 horas;
8- salva as tarefas calculadas na opção 7 em um arquivo "arqDiarias.txt".
